### ¿Como puedo escribir comentarios en JSX? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* El comentario va aquí */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* ¡También funciona
  para comentarios multilínea! */}
  Hello, {name}! 
</div>
```
